# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

### Positive Behavior

- Being welcoming and inclusive
- Being respectful of differing viewpoints
- Gracefully accepting constructive criticism
- Focusing on what's best for the community
- Showing empathy towards others
- Helping beginners learn (that's literally why this project exists!)

### Unacceptable Behavior

- Trolling, insulting comments, or personal attacks
- Harassment of any kind
- Publishing others' private information
- Other conduct which could reasonably be considered inappropriate

## Scope

This Code of Conduct applies to all project spaces, including:
- GitHub issues and pull requests
- Project discussions
- Any other forums created by the project

## Enforcement

Instances of unacceptable behavior may be reported to the project maintainers. All complaints will be reviewed and investigated promptly and fairly.

Project maintainers have the right to remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.
